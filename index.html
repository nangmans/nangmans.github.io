"Passion"

뛰어난 Web Developer란 자신이 하는 일에 열정과 신념을 가진 사람이라고 생각합니다.

수 많은 프로젝트와 활동을 수행하며 제 업무에 열정을 다할 때 최상의 결과물이 나온다는 것을 경험했습니다.





"Match"

개발은 결국 "팀 프로젝트"의 연장선이라고 생각합니다.

코딩만이 중요한 것이 아닙니다.구성원들과 조화롭게 개발하려는 사고방식 또한 중요한 능력입니다.





"Patient"

개발자가 갖추어야 할 가장 필수적인 덕목은 "인내심"이라고 생각합니다.

문제 상황에서 타협하지 않고 해결하려는 의지로 개발에 임해야 할 것입니다.





제가 보유한 기술들을 소개합니다.





Framework / OS

Django

웹 프로젝트(크롤링을 활용한 쇼핑 정보 제공 웹 개발)에서 Django를 활용한 서버 구축 및 관리 경험이 있습니다.

Node.js

웹 프로젝트(커뮤니케이션을 중시한 이미지 업로드 플랫폼 개발)에서 Node.js와 EXPRESS 를 활용한 서버 구축 및 관리 경험이 있습니다.

Ubuntu

Ubuntu 환경에서 개발했던 경험이 다수 있으며, Ubuntu에서 개발을 하는데 필요한 명령어, 구조 등 지식을 보유하고 있습니다.

Languages

HTML

웹 애플리케이션 프로젝트 시 HTML 문법을 다수 사용한 경험이 있습니다.

CSS

프로젝트 시 CSS 프레임워크인 Bootstrap4 활용한 경험이 있습니다.

JAVASCRIPT

웹 프로젝트 중 JavaScript를 사용한 웹 기능 구현 경험이 다수 있습니다. Node.js+EXPRESS를 사용한 프로젝트 중에는 대부분의 개발 언어를 JavaScript로 택했던 경험이 있습니다.

PYTHON

웹 프로젝트(크롤링을 활용한 쇼핑 정보 제공 웹 개발) 에서 Django서버 구축 언어로 Python을 사용한 경험이 다수 있으며, 알고리즘 풀이 주력 언어로 Python을 주로 사용했습니다.

DataBase

MongoDB

NoSQL 중 MongoDB를 웹 프로젝트에 활용한 경험이 있습니다. NoSQL 스키마 작성 및 질의문 작성 경험이 있습니다.

MySQL

RDBMS 중 MySQL 을 사용한 웹 프로젝트 경험이 있습니다. MySQL 쿼리문 작성, ERM 작성, 스키마 작성 및 데이터 모델링 경험이 있습니다.

Cloud / Co-op

AWS

AWS -SAA 자격증을 취득하며 AWS가 운영하는 서비스에 대한 지식 및 실습 경험이 다수 있습니다. Lamba를 통한 서버리스 코드 실행 경험, RDS를 통한 DB 인스턴스 관리, S3를 통한 버킷 관리, EC2를 통한 서버 구축, Elastic beanstalk를 통한 애플리케이션 배포 및 관리 경험이 있습니다.

Git-hub

Git-hub를 통해 git 이력 관리, 이슈 관리, 코드 리뷰, 코드 관리를 시행한 경험이 있습니다.
